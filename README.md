# MyTodoList

TODO list applicacion for WITRAC interview test

## Getting Started

### Installation

* Clone Repository

```
git clone https://github.com/AlejandroSenior/mytodolist.git
```

* Install dependencies

```
npm i
```

### Executing program

* Run local server by executing 

```
npm run serve
```

* Navigate to

```
http://localhost:8080
```
